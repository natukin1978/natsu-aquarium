# 🌊 Pixel Panorama Aquarium

8ビット調のレトロな生き物たちが、時間の移ろいとともに姿を変える海の中をゆったりと泳ぐ、ブラウザベースの放置型パノラマアクアリウムです。

🎥 サンプル 🎞️

https://natukin1978.github.io/natsu-aquarium/

## ✨ 主な機能とこだわり

### 🐟 生き生きとした生態系

多種多様な生き物: 軽快に泳ぐ魚（Swimmers）、ゆらゆら漂うプランクトン（Drifters）、砂底を歩くカニやエビ（Crawlers）。
くらげの「拍動」シミュレーション: 頭頂部を起点とした伸縮アニメーションと、水の抵抗を感じさせるわずかな回転、そして夜に美しく光る発光エフェクト。
パーソナルスペース: 魚同士がぶつからないよう、互いに距離を保ちながら泳ぐ回避アルゴリズムを搭載。

### 🌅 ダイナミックな環境変化

時間の移ろい: 数分かけて「昼の明るい青」から「夜の深い紺碧」へ、海の色がリアルタイムに補間され変化します。
光の演出: 昼間は水面から差し込む光の筋（God Rays）が揺らぎ、夜になると静かに消えていきます。
環境エフェクト: 絶えず上昇する気泡、時折どこかで発生する気泡バースト、そして深海の静寂を演出するマリンスノー（浮遊物）。

### 🌿 奥行きのあるパノラマ構図

多重レイヤー構造:
背景（海・光の筋）
奥のデコレーション（小さな岩・珊瑚）
生き物・気泡・マリンスノー
前景のデコレーション（大きな流木・珊瑚）
最前面の水草（生き物を遮るように揺れる）
重なり回避配置: 珊瑚や岩などは、ランダムながらもお互いが被らないように最適な距離を保って配置されます。

### 🛠 技術スタック

React + TypeScript: 型安全なコンポーネント管理。
HTML5 Canvas API: 高速なピクセル描画とアニメーション。
数学的アニメーション: サイン波やべき乗を用いた、画像素材に頼らない生命感のある動きの実現。

## 🚀 セットアップ

Bash
```
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

## 📜 フォルダ構成

src/Aquarium.tsx: メインのレンダリングループと状態管理。
src/drawUtils.ts: 背景、砂、水草などの描画ロジック。
src/JellyfishLogic.ts: くらげ専用の特殊な拍動・移動アルゴリズム。
src/constants.ts: 生き物の出現比率や環境設定値。
src/types.ts: プロジェクト全体の型定義。
